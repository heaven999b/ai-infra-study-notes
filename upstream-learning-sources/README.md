# Upstream Learning Sources

这个目录不是随手堆上来的完整 clone，而是从本地 `sources/` 里筛出来的“学习资料快照”。

## 保留了什么

- Markdown 文档
- 源码文件
- 配置文件
- notebook
- 一小部分关键图片和图示

## 删掉了什么

- Git 历史目录 `/.git`
- GitHub 工作流和仓库维护噪音
- 大体积演示动图、视频、数据库、样例附件
- 明显不是学习主线所必需的缓存类内容

## 使用建议

- 如果你只是想学框架和路线，先看根目录笔记。
- 如果你想继续看“原始材料长什么样”，再看这个目录。
- 如果你要跑项目，优先把这里当成“读代码资料库”，不要默认每个项目都值得本地装一遍。
